Sample for Testing

What is this Project?

This is a informal demo web application designed to support automation testing training.

It provides a platform for showcasing fundamental operations and interactive features, helping simulate real-world testing scenarios in a controlled environment.

This project use just HTML, CSS and JavaScript (static files) to easy deployment on github pages.

How to Get Started?

Run Locally

  • Clone this repository on your machine. Clone Repository
  • Run a local server in the project directory the way you prefer.
  • Open the index.html file in your browser.
  • Make changes to the files and refresh the browser to see the updates.

Deploy on GitHub Pages

  • Fork this repository on GitHub. Fork Repository
  • Enable GitHub Pages on repository settings.
  • Choose the main branch as the source.
  • GitHub Pages will run the action on a new push to the main branch.
  • Your site will be available at https://<your-username>.github.io/<repository-name>/.
